# bus-ticket-booking-system
# 🚌 Bus Ticket Booking System
A desktop-based GUI application built with Python and Tkinter. This system allows users to select travel routes, specify the number of seats, and generate an instant digital receipt.

# 🚀 Features
User-Friendly Interface: Built with Tkinter and ttk for a native look and feel.

# Dynamic Calculations: Automatically calculates the total price based on a fixed rate of 500 BDT per seat.

Route Selection: Pre-configured dropdown menus for major cities (Dhaka, Mymensingh, Chittagong, Sylhet).

Instant Receipt: Generates a formatted text receipt within the application window.

Error Handling: Basic validation to ensure names and seat numbers are provided before booking.
# 📖 How to UsePassenger Name: Enter the name of the traveler.Route Selection: Select the "From" (Source) and "To" (Destination) cities from the dropdown menus.Seats: Enter the number of seats you wish to book.Book Ticket: Click the green Book Ticket button.View Receipt: Your ticket details and the total price will appear in the "Ticket Receipt" box at the bottom.

# 🧮 Pricing LogicThe system uses a constant variable for pricing:Price per Seat: 500 BDTFormula: $Total = Seats \times 500$
